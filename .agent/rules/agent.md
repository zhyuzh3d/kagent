请按需执行以下步骤：

1. **阅读 Agent 核心定义**
   - 使用 `view_file` 工具读取 `AGENTS.md` 文件。
   - 严格遵循该文件描述的 Agent 角色职责及行为准则。

2. **检索并应用对应的 Skill 规则**
   - 使用 `list_dir` 工具检查 `.codex/` 及其子目录（例如 `.codex/skills/` 等）。

3. **查阅业务与技术文档**
   - 若遇到业务概念或需要核实架构细节，使用 `list_dir` 检查 `doc/` 目录结构，并读取所需文档。

4. **保持同步**
   - 执行任何代码修改前，确保你的行为受最近拉取的以上三种文件所指导。不需要在这个 workflow 文件中修改或维护规则本身，直接遵循拉取到的动态内容即可。
